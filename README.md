# MediFlow  
**Frontend Interview Task — Doctor-Patient Landing Page**  

Responsive landing page for a doctor-patient management system. All 7 core requirements are fully implemented.  
🌐 Links

Live Demo: https://doctor-patient-management-three.vercel.app
 — view the landing page live

GitHub Repo: https://github.com/asadatik/doctor-patient-landing
 — full source code

## ✨ Features
- Sticky navbar with mobile menu  
- Hero section with dual CTAs  
- 4-feature grid highlighting key functionalities  
- Doctor showcase with tabbed sections  
- Call-to-action signup form  
- Footer  
- Fully responsive design for all devices

## 🚀 Setup & Run
```bash
# Clone the repository
git clone https://github.com/asadatik/doctor-patient-landing.git

# Move into project directory
cd doctor-patient-landing

# Install dependencies
npm install

# Start development server
npm run dev

```
🛠️ Tech Stack

Next.js — React framework for server-side rendering and routing

JavaScript (ES6+) — core scripting

Tailwind CSS — utility-first styling

Framer Motion — smooth animations and transitions
